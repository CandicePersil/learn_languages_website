Unix commands

Utiliser les images flotantes pour présenter les terminaux avec des essais de codes. Présenter les codes réalisés car il faut bien pratiquer. Pour cela il faudra écrire :
< img src="image.jpg" class="float_image" alt="Float_image" />
La classe devra être réutilisée dans le .css en écrivant:
.float_image
{
float: left (or right);
}
Si l'on veut que le texte continu en dessous, il suffit de par exemple donner un attribut class="under" et de l'utiliser dans le .css:
.under
{
float: both;
}
