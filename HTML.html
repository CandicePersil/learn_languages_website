Lets start the css (Cascading Style Sheets) part now :>, the language that will make our website beautiful.
First we can check the compatibility by showing compatibility tables of HTML and CSS on different WebBrowsers. Go to the website named Caniuse.
We will learn to use css by using only a .css file and not the other technics that are said to be of less quality. So, first you need to create this file (name is style.css). Then, you will have to "call" this file in you html file to be able to use it. For that, you need to write < link rel="stylesheet" href="style.css" /> in the header. Now, you can use your file :). You can for example change the color of your texts. You did to reference you text so you css file knows which text needs to have a different color. It is written that way:

It is also possible to change the police used for your text. Personaly, I used "Comic Sans MS" as police for all my texts. For that I wrote:
*/*reference to everything in the page*/
{
font-family: "Comic Sans MS", Verdana, sans-serif;
}
As you surely noticed, I wrote Verdana and sans-serif. That is because it is possible that the police you have chosen is not taking into account by your WebBrower. That why you can precise a second police and if it doesn't work a kind of default police. About images, it is possible to change their position. We actually can make float images.
For that, we have to write:
< img src="image.jpg" class="float_image" alt="Float_image" />
class float_image will be used in.css as a reference point:
.float_image //do not forget to put a point before class name
{
float: left (or right);//the following text is written on the right(or left) of the image
}
Text can also be written under the chosen image by using both in css file:
.under
{
float: both;
}
